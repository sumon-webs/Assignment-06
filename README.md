# Project Description

## 📌 Project Name: DigiTools – Premium Digital Tools Marketplace

## 📖 Description: DigiTools is a modern and responsive web application where users can explore and purchase premium digital tools.

## ⚙️ Technologies Used: 
    -React.js
    -Tailwind CSS
    -DaisyUI
    -JavaScript (ES6+)
    -React Toastify
    -JSON (for product data)

## 🚀 Project Features:
    1. 🔄 Toggle Button
        -Initially Products button is selected and product items are displayed.
        -Clicking Cart button will show cart items.
        -If there are no items in the cart, it will display: "Cart is empty".

     2. 🛒 Real-time Navbar Cart Count
        -After clicking Buy Now on a product, the cart count updates automatically in the -navbar.
        -Removing an item from the cart also updates the count in real-time.

     3. 🔔 React Toastify Notifications
        -Clicking Buy Now on a product shows a success alert.
        -Clicking Remove in the cart shows a warning alert.
        -Clicking Proceed to Checkout shows a success alert and clears the cart.
